## About Zorara

Zorara is a free Roblox script executor made for Windows users who want a simple way to run Lua scripts inside Roblox. It attaches directly to Roblox processes so you can inject custom scripts and automate gameplay tasks without paying for a subscription or entering a license key. The app is designed to be easy to pick up, with an interface that works immediately after installation.

Alongside the executor, Zorara includes an integrated script hub with hundreds of community-picked scripts for widely played Roblox experiences. From automation tools to visual tweaks and gameplay changes, the hub offers a searchable collection that refreshes automatically. Its lightweight design keeps resource use low while staying responsive during longer play sessions.

## FAQ

**Is Zorara safe to use?**
Zorara works by interacting with Roblox processes, which can conflict with the platform's terms of service. Use it at your own risk and make sure you understand the possible consequences.

**Will Zorara work with the latest Roblox update?**
The auto-update engine is intended to keep Zorara aligned with current Roblox releases. In some cases, major platform changes may need a short adjustment window.

**How does Zorara compare to paid executors?**
Zorara includes many of the same capabilities you would expect from premium executors, such as a large script hub and debugging tools, while remaining free and requiring no activation key.

**Can my Roblox account get banned?**
Any third-party script executor can put an account at risk of suspension. For testing, it is best to use alternate accounts.

**Where are my scripts stored?**
Scripts you load, along with your settings, are stored locally in the Config folder inside the application directory.
